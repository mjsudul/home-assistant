# ESPHome Configuration for Sonoff S31 (ESP8266)

The sonoff-s31-esp8266-device-config-template.yaml file is customizable ESPHome configuration file for the **Sonoff S31 smart plug**, which uses an **ESP8266 microcontroller**. This configuration enables power monitoring, relay control, Wi-Fi status reporting, and integration with Home Assistant.

The main configuration file is a YAML template designed for use with the [ESPHome add-on in Home Assistant](https://esphome.io/guides/getting_started_hassio.html). It includes:

- Power monitoring via the CSE7766 chip
- Relay control and button input
- Wi-Fi signal reporting
- Daily and cumulative energy tracking
- Status LED and device status monitoring
